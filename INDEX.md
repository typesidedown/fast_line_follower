## üì± Quick Command Reference

### Maze Control
```
Button 1 (Pin 2)          ‚Üí Start DRY RUN
Button 2 (Pin 3)          ‚Üí Start FINAL RUN
```

### Serial Commands
```
help                      ‚Üí Show all commands
maze_status               ‚Üí Show maze state
kp=<value>                ‚Üí Tune proportional gain
kd=<value>                ‚Üí Tune derivative gain
speed=<value>             ‚Üí Set forward speed
thresh_0=<value>          ‚Üí Calibrate IR sensor
dist_calibrate            ‚Üí Calibrate accelerometer
```

### Display Modes
```
IDLE                      ‚Üí Menu, waiting for input
DRY RUN                   ‚Üí Live exploration progress
FINAL RUN                 ‚Üí Optimized path execution
```
